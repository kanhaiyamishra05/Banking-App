📖 Description
The Banking Management System is a console-based Java application designed using core Object-Oriented Programming (OOP) principles. This project simulates basic banking operations such as account creation, deposit, withdrawal, balance inquiry, and transaction management.

It demonstrates how real-world banking systems can be modeled using classes, objects, inheritance, encapsulation, abstraction, and polymorphism.

🚀 Features
Create a new bank account
Deposit money
Withdraw money
Check account balance
Transaction history
Secure login system (optional enhancement)
Input validation and error handling
🛠️ Technologies Used
Java
OOP Concepts
JDBC (optional, if database is used)
MySQL (optional)
🧠 OOP Concepts Implemented
Encapsulation – Data hiding using private variables and public methods
Inheritance – Reuse of code for different account types
Polymorphism – Method overriding for transaction behaviors
Abstraction – Abstract classes/interfaces for banking operations
